# PlanTick 规格说明

## 1. 产品概述

PlanTick 是一个面向单用户的待办事项与日历网页应用，采用 `Web + PWA` 形式交付。

目标使用场景：
- 在 Windows 电脑浏览器中使用
- 在手机浏览器中使用
- 在桌面和手机上以 PWA 形式安装

V1 的核心目标：
- 提供简单、稳定的待办事项管理
- 在月视图日历中展示待办截止日期
- 支持手动创建日程
- 在多设备之间同步数据
- 不做显式账号系统
- 优先降低开发和维护复杂度

## 2. 用户与设备

目标用户：
- 单一用户，仅自己使用

目标设备：
- Windows 电脑
- 手机浏览器 / PWA

关键场景：
- 在任意设备上创建、编辑、完成待办
- 在月视图中查看待办和日程
- 在多个设备间保持同一份数据
- 断网时本地可继续使用，联网后自动同步

## 3. 范围

### 3.1 V1 包含内容

待办功能：
- 创建、编辑、完成、删除待办
- 设置分类、截止日期、备注、重复规则

日历功能：
- 月视图
- 在截止日期当天展示待办
- 手动创建、编辑、删除日程

同步功能：
- 本地优先写入
- 联网后同步到云端
- 通过 Supabase 在多设备之间共享数据
- 以前台补拉为主，实时订阅为增强项
- 冲突规则为最后更新时间覆盖

接入方式：
- 不提供注册/登录页面
- 用户通过“工作区口令”进入同一份数据
- 底层可使用匿名会话支撑数据访问，但不暴露为账号系统

### 3.2 V1 不包含内容

- 多用户协作
- 显式注册与登录
- 周视图和日视图
- 系统级后台常驻同步
- 系统级提醒
- 拖拽改期
- 复杂重复规则
- 原生鸿蒙应用与原生 Windows 应用

## 4. 功能需求

### 4.1 待办

每个待办包含：
- 标题
- 分类
- 截止日期
- 完成状态
- 备注
- 重复规则
- 更新时间
- 软删除标记

支持操作：
- 创建
- 编辑
- 删除
- 完成
- 恢复未完成
- 按分类筛选
- 按日期查看

重复规则：
- 按天重复
- 按周重复
- 按月重复
- 完成当前实例后生成下一实例
- 下一实例由服务端统一派生，避免多设备重复生成

### 4.2 分类

每个分类包含：
- 名称
- 颜色
- 更新时间
- 软删除标记

支持操作：
- 创建
- 编辑
- 删除

### 4.3 日程

每个日程包含：
- 标题
- 日期或时间范围
- 备注
- 更新时间
- 软删除标记

支持操作：
- 创建
- 编辑
- 删除

### 4.4 日历

展示模式：
- 月视图

展示规则：
- 待办按截止日期显示在对应日期格中
- 手动创建的日程显示在对应日期
- 同一天允许展示多个待办和多个日程

## 5. 技术方案

### 5.1 前端

前端主线技术：
- `React`
- `TypeScript`
- `Vite`
- `vite-plugin-pwa`

前端职责：
- 提供桌面与移动端统一界面
- 支持安装为 PWA
- 负责本地缓存、离线访问和同步状态呈现

### 5.2 本地存储

本地存储采用：
- `IndexedDB`

本地保存内容：
- `todos`
- `categories`
- `events`
- 本地同步 outbox
- 工作区信息与同步状态

### 5.3 云端

云端方案采用：
- `Supabase`

云端职责：
- 保存工作区数据
- 提供匿名会话
- 提供数据库能力
- 提供 `Edge Function` 处理工作区接入与同步写入
- 视需要提供实时订阅能力

### 5.4 同步模型

- 本地优先
- 客户端先写 IndexedDB
- 再将变更写入 outbox
- 联网后推送到 Supabase
- 另一设备以前台补拉为主获取更新，实时订阅为增强项
- 冲突规则采用最后更新时间覆盖
- `updated_at` 以服务端时间为准

## 6. 数据访问模型

### 6.1 工作区口令

- 用户首次使用时设置一个工作区口令
- 其他设备输入同一口令后进入同一工作区
- 用户层面无账号系统
- 口令校验通过 `Edge Function` 在服务端完成

### 6.2 匿名会话

- 底层允许使用匿名会话或类似机制完成设备级访问
- 该机制仅用于支撑数据读写与同步
- 不向用户暴露为注册/登录流程
- 匿名会话主要用于调用受限的 `Edge Function` 与完成工作区内数据访问

## 8. 非功能要求

- 界面需同时兼顾桌面端与移动端
- 允许离线查看和本地编辑
- 联网恢复后能自动同步
- 刷新页面后本地数据不丢失
- PWA 可安装
- 中文文本显示正常
- 常规操作流畅

## 9. 验收标准

功能验收：
- 可以创建、编辑、完成、删除待办
- 可以创建、编辑、删除日程
- 月视图可正确显示待办与日程
- 可按工作区口令进入同一份数据

同步验收：
- 一台设备修改后，另一台设备能看到更新
- 断网时本地可继续编辑
- 恢复联网后变更可同步
- 并发修改时按最后更新时间覆盖

平台验收：
- Windows 浏览器可正常使用
- 手机浏览器可正常使用
- 两端都可安装为 PWA
